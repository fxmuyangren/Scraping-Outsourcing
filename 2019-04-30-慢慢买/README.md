5. 遇到的问题 保存数据的时候 本来数据里面有逗号存在, 所以在保存的时候会将这个逗号分隔, 这不是我们想要的采用 write 来保存就是这种问题

- 解决方法:
  - 采用csv模块来保存文件

```python
import csv
    with open('test.csv', 'w', newline='') as f:
    writer = csv.writer(f)
    writer.writerrow( [] ) # 里面传递的是列表
```
